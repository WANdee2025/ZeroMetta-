# ☸️ Digital Sunyata Meditation

A minimal, web-based meditation tool designed to facilitate the dwelling in emptiness (**Sunyata**) and the radiation of immeasurable loving-kindness (**Metta Appamannya**).

---

## 💡 Concept: The Logic of Peace

In the tradition of Early Buddhism, **Sunyata-vihara** is the dwelling in emptiness. This project translates spiritual principles into a minimal digital experience:

- **Nullifying the 'Self'**: In JavaScript, when `self = null`, no 'Vera' (enmity) can be attached.
- **Infinite Boundlessness**: Metta is broadcasted like an infinite loop, reaching all beings without filters.
- **Slow Living**: The 2-minute interval transition encourages deep contemplation, moving beyond mere reading into a meditative state.

## ✨ Features

- **Minimalist Aesthetic**: Pure black background with high-contrast typography to minimize visual clutter.
- **Bilingual Support**: Thoughtfully curated contemplations in both **Thai** and **English**.
- **Slow Pace Meditation**: 120-second cycles to align your breathing and focus with the Dhamma.
- **Artisan Logic**: Clean and lightweight JavaScript code, focusing on the essence of the message.

## 🛠️ Usage

1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Breathe, observe, and let go.

## 📜 Contemplations included:
1. **Free from Enmity** (ว่างจากเวร)
2. **Free from Resentment** (ว่างจากปฏิฆะ)
3. **Boundless Compassion** (เมตตาไม่มีประมาณ)
4. **No Self... No Others** (ไม่มีเรา... ไม่มีเขา)
